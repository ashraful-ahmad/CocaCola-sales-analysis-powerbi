# 🥤 Coca-Cola Sales Analysis Dashboard (Power BI)

## 📊 Project Overview
This project presents an interactive Sales Analysis dashboard built using Microsoft Power BI. 
The dashboard provides insights into sales performance, profit trends, regional analysis, and customer behavior.
